Comic Crawler

聯絡作者: 
==========================
eight04@gmail.com
	
使用需求:
==========================
Python 3
	
使用方法:
==========================
- 執行comiccrawlergui.py，貼上連結後點 "加入連結" 來新增任務。選取任務後點右鍵可以選擇刪除、置頂、與置底。
- 在Windows上可以執行winlaunch.pyw，不會顯示cmd。
- 因為下載時會根據任務名稱來建立下載資料夾，若有任務名稱相同的情況（ex. 不同主機漫畫名稱相同），後下載的任務會因為已經下載過了而忽略下載。若遇到這種狀況，建議把任務改名（選擇任務後按右鍵->改名）。
- 下載後會檢查圖片格式是否正確

- 第一次執行程式時會在同目錄下產生 setting.ini，可以設定...
	- savepath，設定下載目錄。
	- runafterdownload，可以設定下載完後欲呼叫的程式，會傳入任務資料夾位置。
- zip.bat是預先寫好的Windows批次檔，會呼叫7z命令將檔案壓縮後刪除資料夾（小心！），與runafterdownload配合使用。
- 什麼是7z？它是個又小又快的壓縮、解壓縮軟體︰http://www.developershome.com/7-zip/
